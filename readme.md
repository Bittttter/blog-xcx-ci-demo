### 如何运行项目 ？
```bash
# clone 项目到本地
git clone https://github.com/jinxuanzheng01/blog-xcx-ci-demo.git
# 安装npm包
cd blog-xcx-ci-demo && npm i
# 执行脚本
npm run build 
```
